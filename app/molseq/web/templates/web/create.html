Create new matching run

Create a new matching run by entering a genbank query and GBIF query.

This will do the following:

  1. Download your genbank query and GBIF query results to the database
  2. Iterate over each genbank result and run it through our automated matching algorithm against the GBIF query results to try and automatically match it to the GBIF occurrence
  3. Create a user interface to show you suggested matches, and allow you to manually curate additional matches
  4. Create a download button for you to retrieve the manual and automatic matches
